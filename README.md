# Cronometro
Criando um cronômetro simples de segundos ou minutos em C#. Diferentemente do algoritmo da calculadora, o cronômetro não funciona apenas com a biblioteca "System", ele precisa da biblioteca "System.Threading" também por conta do método "Thread.Sleep()", que força um tempo de descanso antes do computador executar o próximo comando no código, esse descanso é necessário para o usuário poder visualizar o que está acontecendo, no caso a passagem dos segundos e a mensagem de "cronômetro finalizado" ao final, do contrário tudo aconteceria em milissegundos e o usuário não seria capaz de ver, o que destruiria o propósito do código. O código começa chamando o método de um menu simples que pergunta ao usuário quanto tempo ele deseja contar e explica que ele deve digitar "s" para segundos e "m" para minutos, então ele captura a informação digitada pelo usuário, separando o que é o tempo do que é o tipo (minutos ou segundos) com o "Substring", que delimita qual parte da string digitada pelo usuário será armazenada em cada variável, os números no começo ficam armazenados na variável "tempo" enquanto a letra ao final fica armazenada na variável "tipo", e depois tem um multiplicador para converter os minutos em segundos, mas apenas se a pessoa optar por minutos, por isso usamos o "if", e um segundo "if" para sair do código caso a pessoa digite "0" usando um "System.Environment.Exit(0)", do contrário o código não deixaria o usuário sair nunca. Feito tudo isso, o método "Menu" chama o método "Start", que é o responsável pela contagem do tempo de fato, ele tem um laço de repetição "while" que incrementa o tempo de um em um até que o "tempoAtual" (usado para a contagem) seja igual ao "tempo" (que é o limite, escolhido pelo usuário), limpando a cada iteração para não deixar a saída com muita informação e no final limpa o tempo e exibe a mensagem de que o processo foi finalizado, chamando o método "Menu" novamente para reiniciar o processo.
